ServerAuthToolkit

Server-authoritative movement toolkit for Roblox Input Actions and client prediction.

Register movement modules and animations, then call Start on both the server and client. Modules run each simulation step in priority order and replicate state through root-part attributes.

RegisterDefinition​

self: ServerAuthToolkit,
definition: ServerAuthToolkitTypes.MovementModuleDefinition--

Module definition to register

) → ()

Registers a movement module with input actions, default state fields, and a Process function.

Modules run in priority order (lower runs first). Register before Start when possible; if already started, active players are updated immediately. Duplicate names are ignored.
